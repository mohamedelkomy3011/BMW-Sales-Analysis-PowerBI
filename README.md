BMW SALES ANALYSIS DASHBOARD - POWER BI
=======================================

PROJECT OVERVIEW
----------------
This project presents an interactive Power BI dashboard analyzing BMW car sales 
from 2010 to 2024. The report includes breakdowns by region, model, engine size, 
fuel type, transmission type, and car color. The dashboard is fully dynamic and 
supports filtering by year range.

DASHBOARD PAGES
---------------
1. Landing Page
   - Navigation buttons to all report sections
   - BMW-themed visual layout
   - Project title and designer signature

2. Yearly Dynamics
   - Total Sales
   - Total Revenue
   - Average Price
   - Top Selling Model
   - Sales by Region (Treemap)
   - Sales by Engine Size (Donut)
   - Yearly Sales Trend (Area Chart)

3. Model Insights
   - Total Sales
   - Revenue
   - Average Price
   - Sales by Fuel Type
   - Sales by Transmission
   - Sales by Color
   - Top Models List

TECHNOLOGIES USED
-----------------
- Microsoft Power BI
- DAX Measures
- Interactive slicers and bookmarks
- Custom design elements and icons

KEY INSIGHTS
------------
- Regional performance differences across continents
- Changes in sales trends from 2010 to 2024
- Engine size distribution and buyer preferences
- Color and fuel type preferences
- Top models contributing the highest sales

HOW TO USE THE REPORT
---------------------
- Use the year slider at the top of each page to filter the data.
- Navigate using the left-side icons to switch between pages.
- Hover over visuals to view detailed tooltips and breakdowns.
