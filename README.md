# BMW Sales Analysis Dashboard - Power BI

## Project Overview
This project presents an interactive Power BI dashboard for analyzing BMW car sales from 2010 to 2024. 
The report provides comprehensive insights into sales performance by region, fuel type, transmission type, 
and model-level analysis.
